Это калькулятор на андроид, реализованы:
Стабильный интерфейс (стрелки, delete, all clear, красивая визуализация с помощью katex),
функции: сложение, умножение, деление, вычитание, возведение в дробную степень, квадратный корень, перевод в станданртную и в десятичную запись для дробей, тригонометрические функции в радианах
парсер + обратная польская запись + подсчет итогового стека (вычисление выражение).
Основной массив кода находится в app/src/main/java/ru/fil/calculator
